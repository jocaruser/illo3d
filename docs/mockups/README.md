# Mockups

Structure mirrors `src/pages/`: one folder per page. Each folder can contain light/dark mockup images and optional notes.

- **HomePage/** — Home / Sales: list, New Sale CTA, table (Date, Description, Client, Price, Profit). Light + dark.
- **InventoryPage/** — Tabs: Filaments, Consumables, Printers; tables + add actions. `inventory-light.png`, `inventory-dark.png`.
- **BudgetPage/** — Calculator form + result block (total cost, markup, suggested price). `budget-light.png`, `budget-dark.png`.

Colour palette: see [docs/design.md](design.md). Use CSS variable names when implementing.
