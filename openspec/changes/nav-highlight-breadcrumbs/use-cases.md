## Overview

This change adds wayfinding: the header shows which section is active, and each main data page shows a short breadcrumb trail. Tests will cover nav state, breadcrumb rendering, and that login stays without breadcrumbs.

## Use Cases to Test

- Active nav highlights current route
- Breadcrumbs appear on Clients page
- Breadcrumbs appear on Jobs page
- Breadcrumbs on Transactions page
- Breadcrumbs appear on Expenses page
- Breadcrumbs appear on Inventory page
- Login page shows no breadcrumbs
- Breadcrumb links navigate correctly
